# Logos des marques (bannière « Ils nous font confiance »)

Dépose ici les logos clients en **PNG transparent** (idéalement **blancs**, ils
s'affichent sur fond sombre), nommés exactement :

```
sephora.png     gymshark.png    kosbiotic.png   nivea.png      speedo.png
reebok.png      garmin.png      amazfit.png     novoma.png     conforama.png
```

- Format **PNG à fond transparent**, logo **blanc** (ou clair).
- Hauteur d'affichage ~32 px, largeur automatique : un PNG large (≥ 400 px) suffit.
- Tant qu'un fichier manque, la bannière affiche **le nom de la marque en texte**
  (repli automatique) — donc rien ne casse.

Une fois les fichiers ajoutés et poussés, les logos remplacent le texte
automatiquement dans la bannière animée.
