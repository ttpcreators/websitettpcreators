# Photo section « À propos »

Dépose ici la photo affichée dans le bloc **« Une agence pensée comme un studio »**.

Nom de fichier exact attendu :

```
equipe.jpg
```

- Format : **JPG** (ou renomme en `.jpg`), portrait ou carré idéalement.
- Ratio conseillé : vertical / carré (le bloc fait ~440 px de haut, l'image est
  recadrée en `object-fit: cover`, centrée sur le haut).
- Tant que le fichier est absent, le bloc affiche le **placeholder**
  `[ photo équipe / studio ]` (repli automatique, rien ne casse).

Une fois `equipe.jpg` ajouté et poussé, la photo remplace automatiquement
le placeholder.
